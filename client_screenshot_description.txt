Top Left Window: Linux console on General at ASU through SSH Secure Shell.
Top Right Window: Appearance of html file from commercial website.
Lower Right Window: Source code of html file from commercial website.

Client is build using 'make httpget'. Client is called with './httpget http://www.brainjar.com/java/host/test.html' which will connect to the website htt://www.brainjar.com and attempt to download the file test.html. The client connects to the website, gets the header line of the HTTP response message, and read and writes the file.
Using 'cat test.html' the contents of test.html are verified to match the source code view in the browser.

Commands used:
make httpget
./httpget http://www.brainjar.com/java/host/test.html
cat test.html
